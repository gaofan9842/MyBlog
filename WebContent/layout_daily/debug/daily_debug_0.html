MySql 报错 2003：ERROR 2003: Can't connect to MySQL server on localhost (10061) 解决办法：

出现这个错误的原因是因为本地关闭了MySQL服务，只要开启对应的服务即可。

 

win10下搜索“服务”或“管理工具”。

 

 

进入“服务”面板：

 

 

找到MySQL并右键开启即可。

 

 

测试MySQL服务已经可以正常启动。
